dgemm
----------------------------------
dgemm(transa, transb, m, n, k, alpha, a, lda, b, ldb, beta, c, ldc)
https://software.intel.com/en-us/node/468480
Double GEneral Matrix Multiply

transa			op(A)
transb			op(B)
m
n
k
alpha
a
lda				leading dimension a
b				leading dimension b
ldb
beta
c				result
ldc				leading dimension c

C := alpha* op(A) * op(B) + beta * C
alpha and beta are scalars
A B and C are matrices
op(A) is an m-by-k matrix
op(B) is a k-by-n matrix
C is an m-by-n matrix

op:
'N' or 'n' -> normal
'T' or 't' -> transpose
'C' or 'c' -> complex transpose

dsyev
----------------------------------
syev(jobz, uplo, n, a, lda, w, work, lwork, info)
https://software.intel.com/en-us/node/469176
Double Symmetric EigenVector

jobz
uplo
n        - order of matrix
a        - stores upper or lower triangular part, outputs orthogonal eigenvectors
lda      - leading dimension
w        - Eigenvalues in ascending order
work     - workspace array
lwork    - dimension of w, outputs 0 for success or minimum size of w
info     - indicates the result of calculation

jobz:
'N' only eigenvalues
'V' both values and vectors
uplo:
'U' a is upper triangular
'L' a is lower triangular

work is workspace array
lwork is the size of the workspace array -> affects performance
